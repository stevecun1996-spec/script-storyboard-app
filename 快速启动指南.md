# 快速启动指南

## 📋 启动步骤

### 1. 安装依赖

打开终端（Terminal），进入项目目录，执行以下命令：

```bash
cd "/Users/Zhuanz/Desktop/cursor 工程/剧本到分镜-github/Script_to_Storyboard_Simple-main/Script_to_Storyboard_Simple"
pip3 install -r requirements.txt
```

如果速度较慢，可以使用国内镜像：

```bash
pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 启动应用

#### 方法一：使用启动脚本（推荐）

```bash
cd "/Users/Zhuanz/Desktop/cursor 工程/剧本到分镜-github/Script_to_Storyboard_Simple-main/Script_to_Storyboard_Simple"
./run.sh
```

#### 方法二：直接使用命令

```bash
cd "/Users/Zhuanz/Desktop/cursor 工程/剧本到分镜-github/Script_to_Storyboard_Simple-main/Script_to_Storyboard_Simple"
streamlit run app.py
```

### 3. 访问应用

应用启动后，会自动打开浏览器，或手动访问：

**http://localhost:8501**

## 🎯 使用提示

1. **首次使用**：需要在侧边栏配置 LLM API Key（推荐使用 Deepseek）
2. **输入剧本**：在主界面输入或粘贴剧本内容
3. **生成分镜**：点击"开始分镜"按钮，AI 会自动划分分镜头
4. **编辑完善**：可以编辑每个分镜头的详细信息
5. **导出结果**：一键导出为 Excel 文件（自动保存到桌面）

## ⚠️ 常见问题

### 如果遇到权限错误

尝试使用用户安装：

```bash
pip3 install --user -r requirements.txt
```

### 如果端口 8501 被占用

Streamlit 会自动使用下一个可用端口，查看终端输出的实际地址。

### 如果需要停止应用

在终端中按 `Ctrl + C` 停止应用。
